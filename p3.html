T-BAR 輕鋼架

T-BAR Ceiling System

產品特性 Features

Features

■ 輕鋼石膏天花，目視輕鋼架主十字骨架。
■ 單吊桿 30.35mm 之多樣化設計，符合 CNS1244 標準之要求。
■ 輕鋼架平面度佳，抗拉強度 210MPA。
■ 塗料烤漆厚度為 20–50 μm。

T-BAR is also named ceiling suspension system, ceiling grid, ceiling tee, ceiling profile in different markets. The easy assembly method facilitates fast installation and removal. Its fire-resistance performance is superior to traditional decoration materials such as wood. It supports various ceiling panels including mineral fiber, gypsum board, PVC gypsum board, and metal ceiling panels.

適用範圍 Application

Applications

辦公室、醫院、教堂，以及各種大型的公共場所。

Office, hospital, church, and all kinds of large public spaces.
