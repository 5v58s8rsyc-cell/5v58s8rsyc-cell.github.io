暗架天花系列

Concealed Grid Ceiling Series

產品特性 Features

Features

● 暗架天花板安裝於板上，不會看到骨架，給整個辦公環境或是公共空間更加簡約及舒適。
● 因使用傳統施工方式，現場安裝施工停頓時間短，損失少，且易於施工與維修。
● 使用少量配件，節省人工與材料成本。
● 施工流程明確，安裝過程可以更精準地控制。
● 搭配 3'x6' 或 3'x8' 天花板，工期更精簡。
● 採用 ASTM: C635-00 FURRING BAR SUSPENSION SYSTEM 之升降天花板工程。
● 吊掛配件有：主支撐架、吊桿、L 型邊條、雙邊條、7 字邊條、U 字邊條等。

Concealed Grid is installed on the top of ceiling tile. When looking at the ceiling, we can’t see the ceiling grid. Using fewer accessories saves working time and expenses. More precise installation distance ensures loading capacity. Tile ceiling installation and ceiling panel could both be installed at the same time, easy to control. Furring bar system conforms to ASTM C635-00 standard.
